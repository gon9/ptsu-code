## ファイル命名規則

### E2E評価結果
- フォーマット: `e2e-evaluation-results-YYYYMMDD.md`
- 目的: 複数回の評価を時系列で振り返り可能にする
- 保存場所: `docs/`

### 実装計画
- フォーマット: `N_title.md` (N=実施順序)
- 例: `1_setup-cli-foundation.md`, `2_agentic-loop-and-coordinator.md`
